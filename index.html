Arch · Ollama · Local

Freedom to run AI locally — privacy, power, and control.

SLYME‑OS is an Arch Linux based distribution that bundles local AI tooling (Ollama-friendly) so you can run models on your machine without cloud telemetry. Minimalist, open source, and made for builders.
